## 기능소개
- 교배 Palworld 게임에서 팰을 번식하는 웹 페이지를 구현한 Next.js 컴포넌트입니다. 주요 기능은 다음과 같습니다<br/>
초기에 /api/roadData 엔드포인트에서 데이터를 가져와 화면에 렌더링합니다.<br/>
사용자가 두 개의 팰을 선택하면, 선택된 팰들의 번식 결과를 계산하여 테이블에 표시합니다.<br/>
번식 결과에는 각 팰과 결과의 세부 정보 및 이미지가 포함되어 있습니다.<br/>
드롭다운 메뉴를 사용하여 팰을 선택하고, 번식 결과가 자동으로 업데이트됩니다.<br/>
